# RailFlow - IRCTC High-Conversion Clone

RailFlow is a modern, high-conversion railway booking application designed to streamline the IRCTC ticket booking experience. Featuring AI-powered assistance and a sleek, responsive interface.

## üöÄ Key Features

- **Dynamic Train Search**: Search for trains between stations with real-time-like availability.
- **AI-Powered Support**: Integrated AI chatbot (powered by **Groq Llama 3.3 70B** and **Google Gemini 3 Flash/Pro**) to assist with queries and booking information.
- **Seamless Booking Flow**: Optimized multi-step booking process from passenger details to payment.
- **Modern Dashboard & Analytics**: Visualized booking stats and travel analytics using Recharts.
- **PDF Ticket Generation**: Automatically generate and download professional PDF tickets after successful booking.
- **Premium UI/UX**: Built with React 19 and custom CSS for a fast, responsive, and aesthetically pleasing experience.
